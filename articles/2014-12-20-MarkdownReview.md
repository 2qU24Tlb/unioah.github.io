## Heading
```
#heading level 1
##heading level 2
```
## Font Style
```
*BOLD or _BOLD
**ITALIC or __ITALIC
~~strike through~~
```
## List
```
* or - for unordered item 1
1. ordered item 1
Nest list:
  1. item 1
    * subitem 1
```
## Code Format
```
` for one line
`` ` for multiple lines
`` `python for Syntax highlighting
```

## Link
```
[My Web Site](unioah.me)
```

## Table
```
First Header  | Second Header
------------- | -------------
Content Cell  | Content Cell
Content Cell  | Content Cell
```
